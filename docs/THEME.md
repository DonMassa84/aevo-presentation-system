AEVO Theme System (Repo)

Ziel
Farben sind ein Abruf- und Strukturcode. Nicht dekorativ.

Prinzipien
1) Max 1 Basis, 1 Akzent, 3 Statusfarben (OK, WARN, DANGER)
2) Gleiche Bedeutung auf allen Folien
3) Keine grellen Hintergruende. Text bleibt hell und ruhig.
4) Folientypen klar unterscheiden:
   - Ablauf/Schritte: Accent
   - Risiken/Fehler: Warn oder Danger
   - Ergebnis/Check: OK
   - Prueferfragen: Basis plus Accent, sonst nichts

Themes
- pruefung: niedrigste Reizlast, stabiler Fokus
- training: etwas mehr Aktivierung fuer Ueben
- recall: starker Cue fuer Active Recall, kurze Frage-Antwort-Slides

Tools
- bin/aevo-pack-theme <pruefung|training|recall>
Erzeugt ein eigenes Package mit getinteten Bildassets als ZIP in out/

Hinweis
Tinting ist ein Overlay auf vorhandene Bilder. Es aendert nicht deinen Inhalt.
